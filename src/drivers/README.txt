**********************************************
* Modules necessaires à l'explotation des 
* interfaces ADC et I2C
**********************************************
Pour vérifier les modules chargés taper
# lsmod

1 - Charger le module ADC en tapant la commande ci-dessous
# insmod adc_s3c44.o

2 - Charger le module i2c en tapant la commande ci-dessous
# insmod i2c_s3c44.o

3 - Lancer le serveur NanoX
# ./nano-X &
